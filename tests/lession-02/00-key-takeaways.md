**Các câu lệnh sử dụng git**

- git init
- Liên kết repository vừa tạo với Git: git remote add origin <ssh_link>
- git add .
- git add <filename>
- git commit -m"message"
- git push origin main
- git log
- git status

**Các loại dữ liệu**
- Number
- String
- Boolean
...

**Các loại toán tử**
- +,-,*,/
- So sánh (>,<,>=,<=,==,===)
- Logic (&&,||)
- một ngôi (x++,x--,--x,++x)

**Vòng lặp**
Cú pháp: for (<điều kiện khởi tạo>;<điều kiện lặp>;<cập nhập>){ code}

**Condition**
Cú pháp: if (<điều kiện>){code}
