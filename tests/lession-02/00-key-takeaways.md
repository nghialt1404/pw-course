** Các câu lệnh sử dụng git **

- git init
- git add .
- git add <filename>
- git commit -m"message"
- git push origin main
- git log

** Các loại dữ liệu **
- Number
- String
- Boolean
...

** Các loại toán tử **
- +,-,*,/
